# Профессиональный калькулятор для сайта (арбористика)

Интерактивный калькулятор для расчета стоимости работ по уходу за деревьями (спил, обрезка, удаление пней). Оптимизирован для вставки в Zero Block или HTML-блок Tilda.

## Особенности
- **Адаптивность**: Полностью корректная работа на мобильных устройствах.
- **Вовлечение**: Интерактивные шаги увеличивают время пребывания пользователя на странице.
- **Чистый код**: Нет рекламы, внешних трекеров или лишних зависимостей.

## Как установить на Tilda
1. Перейдите в настройки этого репозитория -> Pages -> установите Source: **GitHub Actions**.
2. После сборки (вкладка Actions) возьмите URL вашего сайта.
3. В Тильде добавьте блок **T123** (HTML).
4. Используйте `<iframe>` для подключения калькулятора:
   ```html
   <iframe src="ВАШ_URL_ЗДЕСЬ" width="100%" height="700px" frameborder="0"></iframe>
   ```

## Разработка
Для локального запуска:
1. `npm install`
2. `npm run dev`
