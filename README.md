# Калькулятор спила деревьев

Калькулятор для расчета стоимости услуг по спилу деревьев, обрезке крон и удалению пней.

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите проект:
   ```bash
   npm run dev
   ```

## Сборка для продакшена

```bash
npm run build
```
После чего файлы из папки `dist` можно загрузить на ваш хостинг или использовать GitHub Pages.
